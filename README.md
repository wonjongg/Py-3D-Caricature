## Usage

Check the basic usage:
```bash
python caricaturize.py --help
```
We provided some sample .obj files. Test on sample meshes:
```bash
python caricaturize.py --outdir=./ --ref=examples/ref.obj --src=examples/src.obj --beta=0.6
```
